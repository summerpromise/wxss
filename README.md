# 打车小程序

一个功能完整的微信小程序打车应用，支持多种出行方式，实时定位，智能价格计算。

## ✨ 功能特点

### 🚗 多种出行服务
- **快车** - 快速便捷的日常出行
- **顺风车** - 经济实惠的共享出行
- **代驾** - 专业的代驾服务
- **机器人** - 智能配送服务
- **无人车** - 未来科技出行体验

### 📍 智能定位
- 自动获取用户当前位置
- 精确到具体建筑物（如：xx大厦）
- 实时显示周边车辆动态

### 🔍 地址搜索
- 关键词智能提示
- 实时地点建议
- 快速选择目的地

### 💰 智能价格计算
- 基于真实驾车距离计算
- 不同车型差异化定价
- 透明的价格规则：
  - 快车：14元起（含3km） + 2.6元/km
  - 顺风车：8元起（含3km） + 1.8元/km
  - 代驾：30元起（含5km） + 4.0元/km
  - 机器人：5元起（含2km） + 1.2元/km
  - 无人车：12元起（含3km） + 2.2元/km

### 🗺️ 地图功能
- 实时地图显示
- 动态车辆标记
- 可拖动调整视角
- 一键重置到当前位置

### 🎨 主题切换
- 白天模式：简约清爽
- 夜间模式：科技感深色主题

## 🛠️ 技术栈

- **框架**: 微信小程序原生开发
- **地图服务**: 腾讯位置服务 API
  - 逆地理编码
  - 关键词输入提示
  - 驾车距离计算
- **UI设计**: 现代化渐变设计
- **动画**: CSS + 微信小程序动画API

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/summerpromise/wxss.git
cd wxss
```

### 2. 配置腾讯地图API Key

1. 访问 [腾讯位置服务](https://lbs.qq.com/)
2. 注册并创建应用
3. 获取 WebServiceAPI Key
4. 在 `miniprogram/utils/map-config.js` 中配置：

```javascript
const TENCENT_MAP_KEY = '你的API_KEY';
```

### 3. 配置微信开发者工具

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入项目
3. 填入AppID（或使用测试号）

### 4. 配置服务器域名

在微信公众平台配置request合法域名：
```
https://apis.map.qq.com
```

**开发调试阶段**可以在微信开发者工具中勾选：
> 详情 → 本地设置 → 不校验合法域名...

### 5. 运行项目

点击"编译"按钮即可在模拟器中预览
