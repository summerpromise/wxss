## 🚀 立即开始

### 第一步：打开微信开发者工具

1. 启动**微信开发者工具**
2. 选择 **"导入项目"** 或 **"打开"**
3. 选择本项目的**根目录**（包含 `project.config.json` 的文件夹）
4. 输入 AppID：`touristappid`（测试用）
5. 点击 **"导入"**

### 第二步：编译运行

1. 等待项目加载完成
2. 点击工具栏的 **"编译"** 按钮
3. 在模拟器中查看效果

### 第三步：真机测试

1. 点击 **"预览"** 或 **"真机调试"**
2. 使用微信扫码
3. 在手机上体验完整功能

## 🛠️ 常见操作

### 切换主题
1. 进入"我的"页面
2. 点击"深色模式"开关
3. 主题立即切换并自动保存

### 测试打车流程
1. 在主页选择一种出行服务
2. 输入上车地址和目的地
3. 查看价格预估
4. 确认行程信息
5. 选择乘客数量
6. 立即下单

### 查看地图
- 主页自动显示地图
- 蓝色标记点显示当前位置
- 地图上的标记显示附近车辆
- 点击右侧按钮可缩放和重定位
