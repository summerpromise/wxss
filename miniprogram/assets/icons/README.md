# 图标资源说明

此目录用于存放小程序所需的图标资源。

## 需要的图标文件：

### TabBar 图标（必需）：
- home.png - 主页图标（未选中状态）
- home-active.png - 主页图标（选中状态）
- user.png - 我的图标（未选中状态）
- user-active.png - 我的图标（选中状态）

### 地图车辆标记图标（可选，可用默认callout）：
- vehicle-taxi.png - 快车图标
- vehicle-robot.png - 机器人图标
- vehicle-autonomous.png - 无人车图标
- vehicle-unavailable.png - 不可用车辆图标

## 图标尺寸建议：
- TabBar 图标：81x81 像素（@3x）或 54x54 像素（@2x）
- 地图标记图标：28x28 像素

## 临时解决方案：
如果没有图标文件，小程序会显示默认样式，不会影响功能。
您可以稍后添加自定义图标来替换。
