# 图标资源说明

## 车辆标记图标

由于小程序使用了动态车辆标记，建议准备以下图标：

### car-marker.png（32x32 像素）
- 用途：地图上显示的车辆图标
- 建议设计：简单的车辆顶视图，带方向指示
- 颜色：蓝色或青色系，与主题配色一致

## 临时解决方案

当前代码使用了占位路径 `/assets/icons/car-marker.png`，如果图标文件不存在，车辆标记将不显示，但不会影响其他功能的正常运行。

## 如何添加图标

1. 准备32x32像素的PNG图标
2. 将图标文件命名为 `car-marker.png`
3. 放置到 `miniprogram/assets/icons/` 目录
4. 重新编译小程序

## 替代方案

如果暂时没有图标文件，可以：
1. 继续使用当前代码（车辆标记可能不可见）
2. 或者修改 `pages/index/index.js` 中的 `generateVehicles()` 函数，移除 `iconPath` 属性，使用默认标记点

```javascript
// 修改这部分代码
vehicles.push({
  id: i,
  latitude: lat,
  longitude: lng,
  type: type,
  direction: Math.random() * 2 * Math.PI,
  speed: 0.0001 + Math.random() * 0.0002,
  // iconPath: '/assets/icons/car-marker.png', // 注释掉这行
  width: 8,  // 使用小圆点
  height: 8,
  rotate: Math.floor(Math.random() * 360)
});
```

这样车辆会显示为默认的红色圆点标记。
